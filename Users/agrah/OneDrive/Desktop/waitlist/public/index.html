About The Algorithm

This project is an AI-powered coding assistant that helps users solve competitive coding problems (CCP) more effectively using a Retrieval-Augmented Generation (RAG) framework.

Instead of scraping or storing copyrighted content, this tool allows users to create their own problem statements, test cases, and constraints. The assistant then provides intelligent, context-aware help tailored specifically to the user's input.
